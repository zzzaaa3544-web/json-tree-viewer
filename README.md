# JSON Tree Viewer

Paste JSON data and view it as an interactive collapsible tree.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Viewer** (or Skip).

**Step 3:** Paste any JSON data into the left panel. The tree renders automatically on the right.

**Step 4:** Click the arrows (&#9660;/&#9654;) to expand or collapse objects and arrays.

**Step 5:** Color coding:
- `Blue` = keys
- `Green` = strings
- `Yellow` = numbers
- `Orange` = booleans
- `Grey` = null

**Step 6:** Use **Expand All** / **Collapse All** buttons to toggle everything at once.

**Step 7:** Click **Load Example** to see a sample JSON with nested data.

## Features
- Real-time JSON parsing and tree rendering
- Collapsible/expandable nodes
- Color-coded value types
- Expand All / Collapse All
- Node count statistics
- Error highlighting for invalid JSON

## Tech
HTML + CSS + JavaScript | No dependencies
